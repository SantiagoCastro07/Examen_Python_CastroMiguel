## Instalación

### Requisitos:
- Python 3.12.10
- visual studio code 1.86.2

### Pasos:
1. Clona este repositorio o descarga los archivo `funciones.py` y `menu_principal.py`
2. No requiere librerías externas.

---

## Uso del proyecto

1. Abre la terminal.
2. Abre la carpeta en donde está el progr
3. Ejecuta el programa:

```

```
